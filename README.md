Projeto: Tabelas Hash em Textos Reais
Disciplina: Estruturas de Dados II

Descrição:
Este projeto implementa e compara uma Tabela Hash (com encadeamento separado) e uma Estrutura Linear (Baseline) para a contagem e busca de palavras em arquivos de texto (tale.txt e quincas_borba.txt).

Como executar:

1. Certifique-se de ter o Python 3 instalado no seu sistema.
2. Coloque os ficheiros tale.txt, quincas_borba.txt e projeto_hash.py na mesma pasta.
3. Abra o terminal (ou linha de comandos) nessa pasta.
4. Execute o seguinte comando:

Bash:
python projeto_hash.py

O script processará os textos e gerará automaticamente o ficheiro resultados_hash.csv com as métricas de desempenho.

Arquivos incluídos:
- projeto_hash.py: Código-fonte com a implementação das estruturas e funções de hash.
- resultados_hash.csv: Dados brutos obtidos nos experimentos.
- README.txt: Este ficheiro de instruções.
